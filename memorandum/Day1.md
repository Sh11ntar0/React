Reactの開発環境を作る
---

### Create React App

```bash
npx create-react-app my-app
cd my-app
npm start
```

> Node.jsをあらかじめインストールしておく必要があります。

参考: [新しい React アプリを作る – React](https://ja.reactjs.org/docs/create-a-new-react-app.html)

コンポーネント思考とは何かを学ぶ
---

### コンポーネントの特徴

- カプセル化されている
- 置換可能である
- 再利用可能である
- コンポーネントを別のコンポーネントに組み合わせて作成可能である
